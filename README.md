# To-Do List App

A simple and interactive to-do list application built using HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Save tasks in local storage

## Technologies Used

- HTML
- CSS
- JavaScript (Vanilla JS)
